Acerca de cursiva-js

About cursiva-js

cursiva-js es un experimento sobre el dibujado de escritura cursiva sobre un elemento <canvas> de HTML.

El código está hecho completamente en JavaScript (ECMAScript 6), sin frameworks o librerías de terceros.

Algunas funciones fueron prototipadas para crear el dibujado con animació en el canvas, siguiendo las fórmulas para curvas descritas en este artículo.

Navegadores soportados

Cualquier navegador moderno debe poder correr la aplicación sin problemas, ya que se utiliza puro código estándar.

Se añadieron algunos polyfills para Internet Explorer 11.

Desarrollo

El código está disponible en https://github.com/israel-munoz/cursiva. Se requiere Node 8.x o superior para correrlo.

Se utilizó Visual Studio Code para el desarrollo.

La aplicación corre sobre un proyecto en webpack, configurado con Babel para convertir a ES5.

Recursos

  • Fuentes Roboto y Roboto Mono, de Google Fonts.
  • Íconos para el pie de página, de Material Design Icons.

cursiva-js is an experiment about drawing script typing on a <canvas> HTML element.

The code is made in pure JavaScript (ECMAScript 6), with no use of third-party frameworks or libraries.

Some prototyping functions were added for creating animated drawing in a canvas, following the formulas for curves as described in this article.

Supported browsers

Any modern browser should be able to run the application with no issues, since it's made with standard code.

Some polyfills were made to fix issues in Internet Explorer 11.

Development

Code is available at https://github.com/israel-munoz/cursiva. It requires Node.JS 8.x or higher to run.

Visual Studio Code was used for development.

The application runs on a webpack project, configured with Babel to convert to ES5.

Resources

  • Roboto and Roboto Mono fonts, from Google Fonts.
  • Footer icons from Material Design Icons.
